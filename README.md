# due diligence | Single Family Investment Property

A single-page investment property financial calculator for evaluating single-family rental investments with rehab cost estimation.

## Features
- Property assessment checklist with scoring
- Market & zoning analysis
- Rehab cost estimator (cosmetic, moderate, full gut)
- Rental income estimator
- Full 10-year DCF model
- Investment summary metrics (cap rate, CoC, DSCR, IRR, NPV, MOIC)
- Excel export

## Design
- Dark theme with refined teal/violet accents
- Inter + Space Grotesk + JetBrains Mono typography
- Subtle dot grid background
- Smooth scroll-reveal animations
